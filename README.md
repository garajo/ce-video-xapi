# Custom Elements Video XAPI

## Run hot loading dev environment

Use this when you're writing new happy code!

```bash
git clone git@github.com:garajo/ce-video-xapi.git
cd ce-video-xapi
npm install
npm run dev
```

visit `localhost:5001` in your browser

## Run preview server

```bash
npm start
```
visit `localhost:5001` in your browser
